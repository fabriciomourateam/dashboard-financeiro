# Dashboard Financeiro - MyShape

Dashboard financeiro interativo para análise de faturamento, clientes e inadimplência.

## Funcionalidades

- Visão geral de faturamento mensal
- Análise de inadimplência
- Relatório de clientes ativos e inativos
- Gráficos interativos com Plotly

## Como usar

Abra o arquivo `dashboard_myshape.html` diretamente no navegador.

Para regenerar o dashboard a partir dos dados:
```bash
python create_dashboard.py
```
